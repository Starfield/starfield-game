Neues Spiel

Der Game-Modus wird gestartet und es erscheint das von Ihnen ausgewählte Puzzle.
Auf dem Spielfeld können Sie Felder ausgrauen und Sterne setzen.
Das Ausgrauen von Feldern dient zur Erleichterung des Spiels und zum Festhalten von eigenen Lösungsmöglichkeiten. Um eine Spielsituation zu sichern, zu der Sie jeder Zeit zurückkehren können, haben Sie die Option Marker zu setzen.
Des Weiteren können Sie zu jedem Zeitpunkt Ihren letzten fehlerfreien Spielstand aufrufen.

 

Feld ausgrauen

Zum Ausgrauen eines Feldes auf dem Spielfeld, führen Sie den Mauszeiger auf das gewünschte Feld und setzen diesen mit einem Rechtsklick.

 

Stern setzen

Zum Platzieren des Sterns auf dem Spielfeld, führen Sie den Mauszeiger auf das gewünschte Feld und setzen diesen mit einem Linksklick.

 

Marker setzen

Um einen Marker zu setzen, klicken Sie auf "Marker setzen".

 

Gespeicherten Marker aufrufen

Wenn Sie zum zuletzt gesetzten Marker zurückkehren wollen, klicken Sie auf "zum letzten Marker zurück".

 

Alle Marker entfernen

Um alle gespeicherten Marker zu löschen, klicken Sie auf "alle Marker entfernen".

 

Fehlerfreien Spielstand wiederherstellen

Um einen fehlerfreien Spielstand herzustellen, klicken Sie auf "Zurück zum Fehler springen".
