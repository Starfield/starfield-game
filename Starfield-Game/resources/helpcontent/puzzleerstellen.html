Puzzle erstellen

Der Edit-Modus wird gestartet und Sie erhalten ein leeres Puzzle der Größe 5x5.
In diesem Puzzle können Sie beliebig Pfeile und Sterne setzen, die Zahlen am Rand werden am Ende automatisch generiert wenn Sie das Spiel starten. Achten Sie allerdings auf die Lösbarkeit des Puzzles die Sie mittels des Buttons abprüfen können.

Pfeile setzen
Zum setzen eines Pfeiles wählen Sie zunächst links in der Toolbar den gewünschten Pfeil mit einem Linksklick aus.

Zum plazieren des Pfeils auf dem Spielfeld, führen Sie den Mauszeiger auf das gewünschte Feld und setzen diesen mit einem Rechtsklick.
Mit weiteren Rechtsklicks können Sie beliebig viele weitere Pfeile des ausgewählten Pfeiltyps setzen.

Pfeile ändern
Zum ändern eines Pfeils wählen Sie einfach den neuen gewünschten Pfeil in der Toolbar aus und überschreiben den zu ändernden Pfeil mit einem Rechtsklick.

Pfeile entfernen
Wenn Sie einen Pfeil entfernen wollen können Sie dies entweder mit einem Rechtsklick auf den ausgewählten Pfeil tun (oder ggf. ein doppelter Rechtsklick wenn ein anderer Pfeil als der auf dem Feld gesetzte Pfeil ausgewählt ist).
Desweiteren haben Sie hier die Möglichkeit das leere Feld (in der Mitte der Pfeile) in der Toolbar auszuwählen um beliebig viele Pfeile auf dem Spielfeld mit einem einfachen Rechtsklick zu entfernen.

Sterne setzen
Einen Stern können Sie einfach mit Hilfe eines Linksklicks dem gewünschten Feld platzieren.

Spielfeldgröße ändern

Um dem Spielfeld die gewünschte Größe zuzuweisen geben Sie bitte in das erste Feld hinter Größe die gewünschte Anzahl der Felder in der Waagerechten ein und in das Feld rechts daneben die gewünschte Anzahl der Felder in der senkrechten ein.
Klicken Sie anschließend auf den Button um die Größe zu ändern.
Sie können auch eines der Felder freilassen um nur die Anzahl der Felder auf einer Achse zu ändern.

Prüfen
Hier können Sie eine Prüfung durchführen, die das Spiel auf Spielbarkeit und Eindeutigkeit überprüft. Hierzu klicken Sie auf
Ein Spiel ist nicht eindeutig, wenn es mehrere Lösungsmöglichkeiten gibt. Dieses Problem kann durch den Einbau von zusätzlichen Pfeilen behoben werden.
Es ist empfehlenswert beim Erstellen eines Puzzles mehrfach zwischendurch zu prüfen, ob das Puzzle lösbar ist, um Fehler früh zu erkennen.
